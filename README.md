# showclicks

A small Windows desktop app that shows visual ripples and a live circle on mouse clicks.
Built with Python, Tkinter, and pynput.

## What it does

* Shows ripple animation where you click
* Circle follows your cursor
* Fully customizable (color, size, speed, opacity)
* Runs in system tray
* Can run on startup

## Requirements

* Windows
* Python 3.9+

## Install

```bash
pip install pynput
```


## Run

```bash
python main.py
```

## Files

* `main.py` – main logic and click listener
* `ui.py` – settings window
* `settings.py` – config handling
* `tray.py` – system tray
* `about.py` – about window
* `autostart.py` – startup logic

## Downloads

Two Windows builds are available:

**Portable EXE** – run directly, no install needed.

**Installer EXE** – installs like a normal app.

You can find them in the Releases section on GitHub
